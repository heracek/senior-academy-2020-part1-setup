### 💾 Terminal: tmux

- **dommand**: `Ctrl + B`
- command + `%`: split left & right pane
- command + `"`: split top & down pane
- command + `Arrow`
- command + `z`: toggle zoom pane
- command + `c`: new window ("tab"))
- `exit` of `Ctrl + D`: kill pane/window
- command + `n`/`p`: previous/next window
- [default key binding](http://man.openbsd.org/OpenBSD-current/man1/tmux.1#DEFAULT_KEY_BINDINGS)
